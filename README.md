# 🌍 EcoPetropavl — Экологический мониторинг

Система мониторинга качества воздуха в Петропавловске с интеграцией Gemini AI для интеллектуального анализа экологических данных.

## ✨ Возможности

- 📊 **Карта загрязнения воздуха** — визуализация PM2.5 и других показателей в реальном времени
- 🤖 **ИИ помощник на Gemini** — интеллектуальные ответы на вопросы об экологии
- 🌡️ **Прогноз погоды** — информация о текущей погоде и прогноз на 3 дня
- 🔴 **Опасные зоны** — выявление и визуализация критически загрязненных участков

## 🚀 Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/eco.git
cd eco
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Настройка переменных окружения

Создайте файл `.env` в корневой папке проекта:

```bash
touch .env
```

или в Windows PowerShell:

```powershell
New-Item -Path ".env" -ItemType File
```

Откройте файл `.env` и добавьте ваш API ключ Gemini:

```env
GEMINI_KEY=your-gemini-api-key-here
```

**Как получить GEMINI_KEY:**

1. Перейдите на [Google AI Studio](https://aistudio.google.com/apikey)
2. Нажмите "Create API Key"
3. Выберите или создайте проект
4. Скопируйте сгенерированный ключ
5. Вставьте его в файл `.env`

### 4. Запуск проекта

```bash
node server.js
```

После успешного запуска вы увидите в консоли:

```
Server running on http://localhost:3000
```

Откройте браузер и перейдите на `http://localhost:3000`

## 🔐 Безопасность

**Важно:** Никогда не коммитьте файл `.env` в репозиторий!

Добавьте `.env` в `.gitignore`:

```bash
echo ".env" >> .gitignore
```

## 📝 Использование

### Веб-интерфейс

- **Карта** — просмотр загрязнения воздуха в реальном времени
- **Боковая панель** — статистика, легенда, погода, информация о проекте
- **Чат-помощник** — задавайте вопросы об экологии, выбирайте быстрые кнопки

### Быстрые действия в чате

- 🔴 **Опасные зоны** — показать участки с опасным загрязнением
- 🌫 **Воздух** — информация о качестве воздуха
- 💡 **Советы** — рекомендации по экологии

## 🛠️ Технологический стек

- **Frontend:**
  - HTML5 / CSS3 / JavaScript
  - Leaflet — интерактивные карты
  - Marked.js — парсинг маркдауна
  - Open-Meteo API — данные о погоде

- **Backend:**
  - Node.js
  - Express.js — веб-сервер
  - Gemini AI API — интеллектуальный анализ

- **APIs:**
  - Gemini API — обработка естественного языка
  - Open-Meteo API — прогноз погоды
  - OpenAQ API — данные о загрязнении воздуха

## 📚 API Endpoints

### POST /ask

Отправить вопрос ИИ помощнику

**Request:**

```json
{
  "prompt": "Какой уровень загрязнения воздуха?"
}
```

**Response:**

```json
{
  "reply": "ответ от Gemini"
}
```

### GET /pollution

Получить данные о загрязнении воздуха

**Response:**

```json
{
  "results": [...]
}
```

### GET /

Получить главную страницу

## 🐛 Решение проблем

### "Cannot find module 'express'"

```bash
npm install
```

### "GEMINI_KEY is undefined"

Проверьте, что файл `.env` создан и содержит `GEMINI_KEY=your-key`

### API недоступен

Проверьте интернет-соединение и валидность API ключей
